Сайт Широківського ліцею №2

Цей репозиторій містить вихідні файли офіційного вебсайту Широківського ліцею №2. Сайт створений для інформування учнів, батьків, педагогів та громади про діяльність закладу освіти, його події, досягнення, ресурси та навчальні матеріали.

Структура проєкту

Папки та файли в репозиторії:

files/ — додаткові матеріали

photos/ — фотографії та зображення

styles.css — основні стилі сайту

graduates.html — випускники

internet.html — інтернет-безпека

nutrition.html — харчування

pozaklasna.html — позакласна робота

psychologist.html — психологічна служба

Як переглянути сайт локально

Склонуйте репозиторій командою:

git clone https://github.com/ВАШ_ЛОГІН/НАЗВА_РЕПОЗИТОРІЮ.git

Відкрийте файл index.html у будь-якому браузері.

Сайт повністю статичний і не потребує додаткових бібліотек чи серверної частини.

Розгортання сайту на GitHub Pages

Увімкніть GitHub Pages:
Settings → Pages → Source → Branch: main, Folder: /root

Після публікації сайт буде доступний за адресою:

https://ВАШ_ЛОГІН.github.io/НАЗВА_РЕПОЗИТОРІЮ/
